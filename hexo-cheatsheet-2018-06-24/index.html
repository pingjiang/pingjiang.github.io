Hexo Cheatsheet
hexo
cheatsheet
发布日期:   2018-06-24

创建一个博客

  1. 使用Hexo命令行工具hexo-cli快速创建一个博客项目。
npm i -g hexo-cli
hexo init blog
cd $_
npm i
  1. 选择一个比较流行的主题Next，并配置
git clone https://github.com/theme-next/hexo-theme-next.git themes/next

这里使用的Pisces风格，可以这样配置。

scheme: Pisces
  1. 配置tags和categories
# tags页面如果需要使用标签云显示，需要设置type: "tags"
hexo new page "tags"
# categories页面如果需要使用标签云显示，需要设置type: "categories"
hexo new page "categories"
  1. 配置静态页面，用于显示demos
# 在source目录下面创建一个demos文件夹，然后配置如下
skip_render:
  - 'demos/**'
  1. 配置数据驱动的页面

在_data目录下面创建json或者yaml文件，然后在代码中可以使用site.data.filename引用文件内容。可以使用swig模板动态创建内容。

  
    
  {% for item in site.data.tools.items %}
    
  • 
      {{ item.name }}
    
    • 
  {% endfor %}

常用命令

# 写一篇新文章
hexo new "post"
hexo new [layout] "post"

# 发布草稿
hexo publish "post"

# 本地预览
hexo server

# clean 默认会开启缓存，所以发布之前最好clean一下
hexo clean

# 发布上线
hexo deploy

常用功能开关

  • 关闭评论 comments: false
  • 引用Gist {% gist gist_id [filename] %}
  • 全屏图片 {% fullimage /image-url, alt, title %}
  • 引用文章 {% post_link slug [title] %}
  • 居中 {% centerquote %}blah blah blah{% endcenterquote %}

参考资料

  • http://theme-next.iissnan.com/
  • https://hexo.io/zh-cn

注意：这里基于Next主题总结，其他模板可能没有某些功能。
